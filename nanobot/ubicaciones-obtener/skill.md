# Obtener Ubicaciones

Esta skill consulta el endpoint GET /api/ubicaciones/ del backend Django.

## Cuándo usar
- Cuando el usuario pregunta "¿dónde están?", "dame ubicaciones", "muéstrame registros"
- Para ver el historial de ubicaciones guardadas

## Implementación
Ejecuta el script `obtener_ubicaciones.py` que hace la petición HTTP.
