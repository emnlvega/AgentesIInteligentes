# Task: Registrar una Nueva Ubicación

## Descripción
Este es un subagente encargado de registrar una nueva ubicación enviando datos a la API Django.

## Pasos a Seguir
1. Obtener los datos de la nueva ubicación a registrar.
2. Enviar los datos a la API Django para su registro.
3. Verificar que la ubicación se haya registrado correctamente.
4. Informar al agente principal sobre el resultado del registro.
